# Brian Lchamari - Full-Stack Developer Portfolio

Welcome to my personal portfolio repository! This project showcases my experience, skills, and featured work as a Full-Stack Developer specializing in high-performance Python/Django ecosystems and modern web platforms.

## Overview

This portfolio is a purely static, high-performance website built with **Vanilla HTML, CSS, and JavaScript**. It was recently converted from a React-based application to prioritize speed, simplicity, and maintainability without sacrificing the modern aesthetic or dynamic interactivity.

### Key Features
- **Responsive Design**: Tailored to look great on any device or screen size.
- **Dynamic Theming**: Includes a custom Dark/Light mode toggle that saves user preference locally.
- **Scroll Animations**: Smooth, lightweight intersection observer animations (replacing heavy JS physics libraries) for elegant section loading.
- **Interactive Elements**: Includes a "Back to Top" functionality and a sticky floating action button for quick WhatsApp connectivity.

## Technology Stack

- **HTML5**: Semantic and accessible document structure.
- **CSS3 & Tailwind CSS**: Tailwind CSS (loaded via CDN) is utilized for robust, utility-first styling, paired with custom CSS for complex theming and keyframe animations.
- **Vanilla JavaScript (ES6+)**: Handles all state logic, intersection observers, and local storage (theme preferences), completely dependency-free.
- **Lucide Icons**: Clean, scalable vector icons loaded via CDN.

## Local Development

Since this is a static site, no complex build tools or dependencies like Node.js are required to run it locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/B-Lchamari/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` directly in your browser, or serve it using a simple local server (recommended for testing module imports or CORS testing). For example, using Python:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000` in your web browser.
